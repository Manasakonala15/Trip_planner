# 🌍 TripPlanner

TripPlanner is a full-stack travel planning web application built using the MERN stack. The application helps users create, manage, and organize trips efficiently through a modern and responsive interface.

---

## ✨ Features

- 🔐 User Authentication (Login & Registration)
- 🧳 Create and Manage Trips
- 📍 Destination Exploration
- 📅 Trip Planning Dashboard
- 🎟 Booking Management
- 👤 User Profiles
- 📱 Responsive User Interface
- ☁️ MongoDB Database Integration

---

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- React Router
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/TripPlanner.git
```

---

### 2️⃣ Navigate to Project

```bash
cd TripPlanner
```

---

# 🚀 Backend Setup

### Navigate to backend folder

```bash
cd backend
```

### Install dependencies

```bash
npm install
```

### Create .env file

```env
PORT=5001
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### Run backend server

```bash
npm start
```

Backend runs on:

```bash
http://localhost:5001
```

---

# 🎨 Frontend Setup

### Open another terminal

```bash
cd frontend
```

### Install dependencies

```bash
npm install
```

### Run frontend

```bash
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

## 🔗 API Connection
